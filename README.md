# WeatherMv
Maldives weather from Maldives Meteorology Service - http://www.meteorology.gov.mv/

## Installation

	composer require aharen/weather-mv

## Basic Usage

	$weather = new WeatherMv('Male');
	$weatherData = $weather->getData();

## Currently Accepted Cities
	
	Male
	Hanimadhoo
	Kahdhoo
	Kaadehdhoo
	Gan
